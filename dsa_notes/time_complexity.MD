# Time Complexity

- Instand - O(1)
- Logrithmic - O(log n)
- Linear - O(n)
- O(n log n)
- Squared - O(n^2)

- Cubed - O(n^3)
  - n <= 500 (10^2-10^3) (2^9)
- Exponential - O(2^n)
  - n <= 12
- Factorial - O(n!)
  - n <= 12 (2^4)

## Bigger than 10^9 (2^30) (1 Billion)

- preferably O(log n)
- O(n) and faster algorithm

**Note**: Most modern CPUs handle roughly 10^8 to 10^9 operations per second.
